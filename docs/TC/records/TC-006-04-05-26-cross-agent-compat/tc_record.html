Overview

Summary

Add install paths and trigger syntax for Cursor, Codex CLI, and Gemini CLI so the TC skill works across all major AI coding agents

Motivation

The skill registries (antigravity, VoltAgent, daymade) all value cross-agent compatibility. Currently the skill only targets Claude Code. Supporting other agents widens the addressable user base and strengthens the registry submissions.

Detailed Design

Map each agent's skill directory convention (.cursor/skills/, .codex/skills/, .gemini/skills/), adapt trigger syntax if needed, add per-agent install instructions to README, and integrate into the npm CLI (TC-004) via --cursor/--codex/--gemini flags.
