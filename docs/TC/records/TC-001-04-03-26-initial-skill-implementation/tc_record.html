Overview

Summary

Complete implementation of the TC (Technical Change) skill for structured code change tracking with JSON records, accessible HTML output, and AI session handoff

Motivation

User manages 10+ projects and needs a universal, bulletproof change tracking system so that when an AI session expires or is abandoned, the next bot can seamlessly resume work. Also serves as human-readable documentation of all code changes.

Detailed Design

7-phase build: JSON schemas first, then Python validator with state machine, accessible CSS/HTML templates, Python generators (stdlib only), init templates for project onboarding, SKILL.md with 8 commands, and example data. Each generated HTML file is self-contained with inlined CSS for file:// compatibility.
